# 软件打开问题记录

## 已观察情况

- 三套软件可以由用户手动打开。
- Computer Use 能列出部分窗口，但对这些老式软件窗口多次返回 `window is not a usable app window`。
- 因此自动读取界面文字不稳定。

## 推荐补录方式

1. 手动打开软件到目录页。
2. 截图或手动转录左侧目录树。
3. 粘贴到 `软件界面补录模板.md`。
4. 核对后同步到对应索引文件。
