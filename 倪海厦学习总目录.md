## 总体学习地图

倪海厦相关资料可以分成四块：

1. 访谈导入
   - 《梁冬对话倪海厦》适合做入门导览。
   - 重点不是直接学处方，而是了解倪海厦对中医、经典、临床和学习方法的整体看法。

2. 人纪
   - 这是医学学习主线。
   - 通常围绕针灸、本草、黄帝内经、伤寒论、金匮要略、八纲辨证、仲景心法、临床案例等内容展开。
   - 本地对应软件为“倪海厦人纪内部学习系统V2023版”。

3. 天纪
   - 偏向易经、紫微斗数、命理、风水等传统术数内容。
   - 与中医学习有关联，但不应放在医学主线前面。
   - 本地对应软件为“倪海厦天纪学习系统客户端V2023版”。

4. 医学论文医案
   - 用于补充病例、论文、医案检索。
   - 适合在人纪主干学完一部分之后，用来做主题复盘和案例对照。
   - 本地对应软件为“倪海厦医学论文医案查询系统V2023版”。

## 推荐学习顺序

### 第 0 阶段：建立边界

目标：
- 明确这是传统中医学习资料，不是自我诊疗系统。
- 先学概念、辨证和经典框架，不急于套方。

建议动作：
- 建一个学习笔记目录。
- 每条笔记标明来源：访谈、人纪、天纪、医案、其他。
- 涉及处方、剂量、针灸操作的内容，只做学习记录，不自行实施。

### 第 1 阶段：访谈导入

资料：
- 《梁冬对话倪海厦》01 到 07

学习目标：
- 了解倪海厦的学习观、经典观、临床观。
- 标记后续需要深入学习的关键词。

建议笔记栏目：
- 关键观点
- 提到的经典
- 提到的疾病或病机
- 与现代医学需要区分的说法
- 后续在人纪中对应查找的主题

### 第 2 阶段：中医基础与辨证框架

资料方向：
- 八纲辨证
- 阴阳、表里、寒热、虚实
- 脏腑、经络、气血津液

学习目标：
- 先能判断一个说法属于哪类辨证框架。
- 避免直接记方而不知道适用边界。

建议输出：
- `八纲辨证.md`
- `脏腑辨证.md`
- `经络基础.md`
- `常见术语表.md`

### 第 3 阶段：针灸

资料方向：
- 人纪针灸相关内容
- 经络、穴位、取穴原则、针灸禁忌

学习目标：
- 先掌握经络逻辑和穴位体系。
- 临床操作部分只做理论学习，实际操作需专业资质。

建议输出：
- `针灸总览.md`
- `十二经络.md`
- `常用穴位索引.md`
- `针灸禁忌与风险.md`

### 第 4 阶段：本草

资料方向：
- 神农本草经
- 药性、归经、禁忌、配伍

学习目标：
- 认识药物分类和药性逻辑。
- 把药物放回辨证系统中理解，而不是孤立背功效。

建议输出：
- `本草分类.md`
- `常用药物索引.md`
- `药性术语.md`
- `配伍与禁忌.md`

### 第 5 阶段：伤寒论

资料方向：
- 六经辨证
- 桂枝汤、麻黄汤、承气汤、柴胡剂、四逆类等经典方

学习目标：
- 建立外感病和六经传变框架。
- 理解经典方的证候边界。

建议输出：
- `伤寒论学习路线.md`
- `六经辨证.md`
- `伤寒方剂索引.md`

### 第 6 阶段：金匮要略

资料方向：
- 杂病、妇科、痰饮、水气、胸痹、虚劳、血证、黄疸等

学习目标：
- 从伤寒的外感框架转入杂病辨证。
- 建立病名、病机、方证之间的对照。

建议输出：

## 当前结论

现阶段最稳妥的学习主线是：

```text
梁冬访谈导入
-> 八纲辨证和基础术语
-> 针灸
-> 本草
-> 伤寒论
-> 金匮要略
-> 医案论文复盘
-> 天纪作为传统文化分支补充
```

这个顺序的好处是先建立医学辨证框架，再进入方药和病例，最后把天纪作为扩展内容处理，避免一开始就把医学、术数和访谈观点混在一起。
