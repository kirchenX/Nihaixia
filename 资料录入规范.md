# 资料录入规范

## 目标

让每条学习笔记都可追溯、可核对、可继续迭代。

## 来源类型

记录资料时先标明类型：

- B 站视频
- 本地软件界面
- 软件说明视频
- 人工转录
- 截图 OCR
- 外部网页或仓库
- 个人总结

## 状态标签

建议使用以下状态：

- `已确认`：已从页面、文件、软件界面或用户提供内容确认。
- `待核对`：看到了线索，但尚未确认完整内容。
- `推导`：根据学习结构整理出的路线或分类，不等同于原文。
- `高风险`：涉及处方、剂量、针灸操作、重症判断、儿童孕产等内容。

## 软件界面补录

如果软件窗口无法自动读取，使用 [软件界面补录模板](软件界面补录模板.md)：

1. 手动打开软件到目录页。
2. 截图或人工转录目录树。
3. 标记不确定文字。
4. 同步到对应索引。

## 禁止事项

- 不上传 `.mdb`、`.ldb`、`.exe`、`.dll`、视频压缩包等原始资料。
- 不把个案写成通用治疗方案。
- 不把访谈观点写成已验证医学结论。
- 不删除原始软件目录内文件。
